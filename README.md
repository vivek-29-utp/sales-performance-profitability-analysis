# Sales Performance & Profitability Analysis

## Overview
An Excel-based business analytics dashboard project designed to analyze sales performance, profitability trends, and customer behavior.

## Features
- KPI tracking dashboards
- Sales trend analysis
- Region-wise sales analysis
- Profitability monitoring
- Interactive slicers and charts

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Dashboards

## Project Structure
- Excel Dashboard File
- Dashboard Screenshots
- Dataset
- Documentation
